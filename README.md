Welcome to Scorpio Games, a full-stack digital distribution platform clone modeled after Steam. This project was developed as a final lab project for the Web Technologies Lab, focusing on modern frontend development with React.js, real-time communication with Socket.io, and persistent data management using MySQL.




Project Overview

Scorpio Games is a robust Single Page Application (SPA) that enables users to browse, purchase, and manage video games in a dynamic environment. The system features:



Interactive Authentication: A secure login/signup system featuring an animated Panda UI that reacts to user input.


Real-Time Live Chat: Instant messaging between friends powered by Socket.io, with permanent history stored in MySQL.




Scorpio Wallet: A functional digital wallet where users can redeem gift card vouchers and purchase games.



Social System: Features to search for friends, send/accept friend requests, and track online status.



Account Recovery: A secure "Reset Password" flow utilizing security questions and Bcrypt encryption.


Tech Stack
Frontend

React (Vite): For a fast, component-based user interface.


Socket.io-client: For bi-directional real-time communication.


CSS3: Custom dark-themed professional styling (Steam-inspired).


React Router: To handle client-side navigation without page refreshes.


Backend

Node.js & Express.js: Server-side runtime and API development.


Socket.io: Real-time engine for messaging and notifications.



Bcrypt: For industrial-standard password hashing.


Multer: Handling profile picture uploads and storage.

Database

MySQL: Relational database for persistent storage of users, libraries, and chats
